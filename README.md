# 项目运行

## 安装项目依赖
npm install

## 打开微信开发者工具
* 目录指向 `dist`
* appid: wxb606392a692583e6

## 微信小程序
npm run dev:weapp

### 开发主要目录说明
    
    - project
    |———— src
    |   |———— assets    //  本地资源目录
    |   |———— config    //  接口配置
    |   |———— models    //  数据管理
    |   |———— pages        //  页面源代码目录
    |   |———— utils     //   公共函数
    |   app.jsx
    |   app.scss
    |   index.html   //  html模板
    |
    |———— dist              // 打包输出目录
